Install downloaded fonts on Windows 10

23/6/2023 | By: saaiqSAS

Method 1

1. Right click on the font file you want to install and click 'Install'. If you have administrator privilege, then you can click 'Install for all users'.

Method 2

1. Open the font file you want to install, and on the window that comes up, click 'Install'.
