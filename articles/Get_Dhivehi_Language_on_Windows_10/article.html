Get Dhivehi Language on Windows 10

23/6/2023 | By: saaiqSAS

In order to type Dhivehi, you need to have Dhivehi language installed on your system. Below are the steps you can follow to achieve this on Windows 10.

1. Click the search button on the taskbar, or you can use the keyboard shortcut Win + S .

2. Now type ‘language’ and click on ‘Language settings’.

3. Now click ‘Add a language’.

4. On the search box that comes up type ‘Divehi’.

5. Select ‘ދިހެހިބަސް’ and click ‘Next’.

6. Now click ‘Install’.

7. Now you should have Dhivehi language installed. To change to Dhivehi language from current language, click the language button in taskbar.

8. Now select ‘Divehi’.

That’s all. Now with the right fonts you will be able to type Dhivehi.
